Anatomy Quiz iOS Application

Anatomy Quiz is an application created for a Mobile App Developement class that tests the users knowledge of various muscles from a selected muscle group.

Landing Page. Click "Yes" to see list of muscle groups. Click "No" to be redirected.

Muscle Groups. Click a muscle group to be quizzed on this muscle group.

Back muscles. Click your answer to see if you're correct.

A pop up will notify you if your answer is correct or incorrect. An incorrect anser will force you to try again. A correct answer will take you to the next question.

Workout Warrior Android Application

Workout Warrior is a workout app that allows users to choose a muscle to workout. Users can view exercises, choose rest periods and add to a database to optimize their workouts.

Landing Page. Click on "Enter" to see list of muscle groups

Muscle Groups. Click a muscle group to see a list of exercises.

Chest Exercises. Click an exercise to see details.

Barbell Bench Press. Options of rest periods, entering into a database, and picture of exercise are outlined here.

GameBetter

Final project for Fundimentals of Software Engineering class from the fall of 2016. This site, GameBetter, is made through PHP and MySQL by a total of 3 contributors. My contributions were the site template code, add game, add comment and rate game functions.

Home Page

Unregistered users can view current games or register as a user. Registered users can login to review, rate, or add new games.

Add New Game

Administrators add new games with various qualities. Options Platform, Rating and Genre have a dropdown menu of pre-determined qualities.

Add Comment

All games are dymanically added to the drop down list. Users select a game, leave a comment, then click "Submit".

Rate Game

All games are dymanically added to the drop down list. Users select a game, select a star rating, then click "Submit". Stars turn yellow when hovered over.

Game Reviews

Reviews of games from administrators are dymanically added here.

User Reviews

Reviews of games by users are dymanically added here.

Current Games

All games are added here with their ratings. Ratings are an average of 1-5 stars.
