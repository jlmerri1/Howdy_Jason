GameBetter

Home Page

Final project for Fundimentals of Software Engineering class. This site, GameBetter, is made
through PHP and MySQL with phpMyAdmin. Unregistered users can view current games.
Registered users can review and rate current games and registered administrators can add new games.

Add New Game

Administrators add new games with various qualities.

Add Comment

All games are automatically added to drop down list. Users select a game, leave a comment, then click "Submit".

Rate Game

All games are automatically added to drop down list. Users select a game, select a star rating,
then click "Submit". Stars turn yellow when hovered over.

Game Reviews

Reviews of games from administrators are automatically added here.

User Reviews

Reviews of games by users are automatically added here.

Current Games

All games are added here with their ratings. Ratings are an average of 1-5 stars.
